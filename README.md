DESHIMA measurement set by DataArray

## Installation

```shell
pip install dems
```

## Usage

```python
from dems.d2 import MS


da = MS.new(
    data=[[0.0, ...], ...],
    mask=[[False, ...], ...],
    weight=[[1.0, ...], ...],
    ...
)
```
